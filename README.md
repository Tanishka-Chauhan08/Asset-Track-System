# Asset Auditor Pro | IT Asset Management System

A simple and practical Asset Management & Compliance tool built using **React** and **Tailwind CSS**. It helps track hardware and software assets, check their compliance status, and manage records in an organized way.

---

## 🌐 Live Demo
[View Live Dashboard](https://asset-track-system.netlify.app/)

---

## 🚀 Key Features

- **Asset Management:** Easily add, view, and remove assets with instant UI updates using React state.
- **Live Compliance Overview:** Displays total assets, compliance percentage, and non-compliant items in real-time.
- **Search & Filtering:** Quickly find assets by name, serial number, or assigned user.
- **CSV Export:** Download asset data as a CSV file for reporting or audit purposes.
- **Data Persistence:** Uses `localStorage` to keep data saved even after refreshing the page.
- **Responsive Design:** Built with Tailwind CSS to work smoothly across mobile, tablet, and desktop screens.

---

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Build Tool:** Vite
- **Storage:** Browser LocalStorage

---

## ⚙️ Concepts Used

- **CRUD Operations:** Add, view, and delete assets.
- **State Management:** Managed data using `useState` and `useEffect`.
- **Responsive UI:** Designed using Tailwind utility classes for different screen sizes.
- **CSV Export Logic:** Implemented file download using JavaScript (Blob API).
